Everything you need. Nothing you don't.

Built for drone pilots, event operators, and anyone who needs to get a live video signal from point A to everywhere else.

🎯

Full-screen projection

Project your stream to any connected monitor or TV at the click of a button. Perfect for displaying drone footage on a big screen at events.

📶

Universal RTMP input

Works with DJI drones, DJI RC controllers, OBS Studio, and any device that can push an RTMP stream. No special drivers or apps needed on the sender.

🌐

Live web player

Broadcast to a password-protected web page that anyone can watch on their phone or laptop — no streaming platform account required. Powered by Cloudflare Tunnel, so no port forwarding needed.

⏺️

Local recording

Record every stream automatically to MP4 on your local drive. Choose which streams to record individually, and set a custom save location.

🔑

Multiple stream keys

Manage multiple stream keys with individual names and settings. Each key can have its own recording and restream configuration.

🔄

Re-stream to platforms

Forward your stream to YouTube Live, Twitch, or any RTMP destination simultaneously, without leaving the app.

🔔

Auto-projection

Optionally project the stream fullscreen as soon as it connects — no need to be at the keyboard. Ideal for unmanned display setups.

🔁

Auto-updates

Checks for new releases on startup and updates silently in the background. You always have the latest version without any manual effort.

How it works

Three steps from launch to live.

1

Install and add your stream key

Download and install RTMP Projector. Add a stream key — the app generates a unique RTMP URL automatically. Point your drone or encoder at it.

2

Start the server and fly

Hit Start Server. The moment your drone connects, the stream appears in the app. Click Project to push it fullscreen to your display monitor.

3

Share the web player (optional)

Run the one-time web stream setup wizard to connect your Cloudflare account. From then on, anyone with the link — and the password if you set one — can watch live from their browser.
